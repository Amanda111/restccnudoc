# 常用网站API

> 获取常用网站

| URL | Header | Method |
|----------- |:--------: |-------:|
| /api/site/ | 无 | GET |

<hr/>

## URL Params

    无

## POST Data(json)

    无

## Return Data(json)

    [
        {
            "site": "网站名称",
            "url": "网站网址"
        }
    ]

## Status Code

    200: 请求成功
