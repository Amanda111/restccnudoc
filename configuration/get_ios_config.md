# 获取IOS json配置API

> 获取ios json配置

| URL |  Header | Method |
| ------------- |:-------------:| -----:|
| /api/ios/config/ | 无 |  GET |

<hr/>

## URL Params

    无

## POST Data(json)

    无
    
## Return Data(json)

    {"config": {'k1':'v1', 'k2':'v2'}}    # ios json配置
    
## Status Code

    成功返回: 200
    配置不存在: 404
